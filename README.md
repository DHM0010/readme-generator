# readme-generator

## Description

the readme-generator is an application designe to automatically create a README.md file for your projects so you dont have to waste time building it manually

## Table of Contents

- [Usage](#usage)
- [License](#license)

## Installation

install nodeJS then just run node index.js from yyour integrated terminal

## Usage

Upon running node.js you will be asked a series of questions, once all the questions are asked the file will automatically be created.
